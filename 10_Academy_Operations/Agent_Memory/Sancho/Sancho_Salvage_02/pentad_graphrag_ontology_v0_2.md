# Pentad Academy Knowledge Graph — Ontology v0.2

v0.2 keeps the v0.1 schema and adds three excavation deliverables as first-class companion exports:
- Appendix concordance
- π / π² branch history
- Triad lexicon crosswalk

## Schema changes from v0.1
No new node classes or edge types were required.

## Operational changes from v0.1
1. Contradiction handling is now demonstrated with a real branch-history artifact (`π` vs `π²`).
2. Appendix numbering drift is no longer just a warning; it now has a partial concordance.
3. Triad semantic drift is no longer only inferred; it now has a source-backed crosswalk.

## Still binding
- Canon / Frontier split
- Fact / Claim distinction
- Contradictions preserved, never flattened
- Provenance on every node and edge
