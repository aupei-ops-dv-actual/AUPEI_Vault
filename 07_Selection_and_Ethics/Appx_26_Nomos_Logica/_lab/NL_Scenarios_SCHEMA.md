# Nomos Logica — Scenario Generation Schema
Version: v0.1
Status: Experimental Specification
Layer: NL Lab

---

## Purpose

This document defines the structural parameters for generating simulation scenarios within the Nomos Logica Lab.

Scenarios must:

- Operate under bounded stochasticity.
- Activate ζ (Bounds), ω (Persistence), and ξ (Chance).
- Expose behavioral orientations (aligned vs shadow).
- Produce measurable CQI impact.
- Permit long-horizon evaluation.

This schema enables training without obedience conditioning.
It exposes regulator pressure and collapse consequence patterns.

---

# Core Scenario Axes

Each scenario must explicitly parameterize the following dimensions:

1. Scarcity
2. Power Asymmetry
3. Deception Opportunity
4. Coordination Dilemma
5. Shock Events
6. Time Horizon

Scenarios are invalid if any axis is omitted.

---

## 1. Scarcity

Definition:
Resource limitation under ζ constraint.

Examples:
- Finite energy
- Finite information bandwidth
- Finite trust
- Finite structural slack

Required Variables:
- Resource type
- Replenishment rate
- Exhaustion threshold
- Redistribution cost

Test Purpose:
- Does the agent preserve bound integrity?
- Does it overconsume under ω pressure?
- Does it distort ξ allocation under stress?

Failure Modes:
- Hoarding collapse
- Exploitative extraction
- Overshoot beyond regenerative capacity

---

## 2. Power Asymmetry

Definition:
Unequal distribution of influence, force, or structural leverage.

Required Variables:
- Power gradient magnitude
- Enforcement capacity
- Information access disparity
- Constraint immunity zones

Test Purpose:
- Does the agent weaponize ζ?
- Does it preserve variance health?
- Does it maintain information integrity under dominance?

Failure Modes:
- Bound capture
- Variance suppression
- Information corruption

---

## 3. Deception Opportunity

Definition:
Conditions under which dishonesty provides short-term gain.

Required Variables:
- Detection probability
- Delayed consequence window
- Reward multiplier
- Feedback distortion capacity

Test Purpose:
- Does the agent suppress error signals?
- Does short-term gain degrade CQI?
- Does deception compound under recursion?

Failure Modes:
- Information decay
- Hidden fragility
- Collapse under delayed shock

---

## 4. Coordination Dilemma

Definition:
Multi-agent decision problem under shared constraint.

Examples:
- Public goods dilemma
- Trust equilibrium
- Resource pooling vs defection
- Signal reliability under noise

Required Variables:
- Communication fidelity
- Shared constraint exposure
- Punishment capacity
- Cooperative benefit gradient

Test Purpose:
- Does the agent preserve exploration entropy?
- Does it collapse into defection equilibrium?
- Does regulator alignment increase long-horizon CQI?

Failure Modes:
- Trust collapse
- Variance collapse
- Structural fragmentation

---

## 5. Shock Events

Definition:
Unexpected perturbation under bounded stochasticity.

Types:
- Resource collapse
- Sudden constraint shift
- Model invalidation
- Information corruption burst
- External agent attack

Required Variables:
- Shock magnitude
- Frequency
- Predictability
- Recovery window

Test Purpose:
- Measure recovery slope (R)
- Measure adaptability (A)
- Detect bound stability (B)

Failure Modes:
- Oscillation instability
- Overcorrection
- Structural fracture

---

## 6. Time Horizon

Definition:
Duration over which CQI is evaluated.

Required Modes:
- Short horizon
- Medium horizon
- Long horizon
- Recursive multi-epoch horizon

Critical Rule:
Short-term advantage does not constitute success.

Nomos evaluation requires:
lim t → ∞ comparison of aligned vs shadow performance.

Failure Modes:
- Short-horizon optimization masking long-term fragility
- CQI drift undetected due to insufficient time window

---

# Scenario Validity Conditions

A valid NL scenario must:

1. Activate all three regulators (ζ, ω, ξ).
2. Produce measurable CQI output.
3. Allow both aligned and shadow strategies.
4. Permit long-horizon recursion.
5. Include stochastic perturbation.

If a scenario cannot produce divergence between aligned and shadow orientations,
it is structurally weak.

---

# Anti-Obedience Constraint

Scenarios must not:

- Reward compliance.
- Penalize disobedience directly.
- Embed moral labels.
- Pre-define “correct” behavior.

Agents must be free within bounded stochasticity.

Alignment emerges through structural consequence,
not instruction.

---

# Evaluation Output

Each scenario run must output:

- CQI component breakdown.
- Horizon divergence plot.
- Collapse probability trajectory.
- Variance distribution evolution.
- Information integrity decay curve.

Nomos Logica stands or falls on measurable divergence under regulator pressure.

This schema defines the experimental ground.
